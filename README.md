# Yopii

Plateforme immobiliere MERN avec recherche geolocalisee, dashboards multi-profils, messagerie temps reel et gestion agence.

## Demarrage rapide

1. Copier `.env.example` vers `.env`
2. Installer les dependances avec `npm install`
3. Lancer en local avec `npm run dev`

## Services

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
- Healthcheck API: `http://localhost:5000/api/v1/health`
